# Enyir Project

This is the main package for the enyir project integration with the Lead Profiling and Enrichment Engine.

## Directory Structure

```
enyir/
├── __init__.py       # Package initialization
├── src/              # Main source code
├── utils/            # Utility functions and helpers
└── data/             # Data files and resources
```

## Usage

Import modules from this package in your Streamlit app:

```python
from enyir.src import your_module
from enyir.utils import your_utilities
```

## Adding Your Code

1. Place your main application modules in `src/`
2. Add utility functions in `utils/`
3. Store data files in `data/`

## Development

Make sure all your Python files have proper docstrings and follow PEP 8 style guidelines.
